# The Cozy Cup

Premium cafe portfolio site built with Vite, React, Tailwind CSS v4, and Motion.

## Stack

- Vite 8
- React 19
- Tailwind CSS 4 via `@tailwindcss/vite`
- Motion for parallax and scroll animation
- Lucide React for icons

## Features

- Cinematic full-page hero with layered parallax surfaces
- Editorial typography and luxury hospitality color palette
- Sticky storytelling sections for brand, menu, and space positioning
- Premium CTA and reservation block for conversion-focused portfolios
- Production build and lint validated

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Notes

- The app lives at the repository root.
- The leftover `cozycup-temp` folder is only an ignored temporary scaffold artifact if it still exists locally.
- CI/CD can be added later without changing the current page structure.
